A lightweight and flexible event registration plugin for WordPress. Perfect for small businesses, venues, nonprofits, and creative teams who need an easy way to manage RSVPs and attendee signups.

== Description ==

Arbe Events is a simple yet powerful plugin that allows you to:

- Create and manage events using a custom post type
- Accept registrations through a frontend form
- Send automatic email confirmations to admins and attendees
- View and export registrant data
- Integrate seamlessly with your site’s theme and style

No bloated features. No short-term hacks. Just clean, scalable event registration built for real businesses.

Perfect for:
- Fitness studios
- Breweries & venues
- Nonprofits
- School events
- Community programs
- And more…

== Features ==

* Custom Post Type: `event`
* Shortcode-based frontend registration forms
* Email confirmation to admin and registrants
* Admin view of registrants
* Export attendees to CSV
* Minimal styling, theme-friendly
* Built for developers and agencies
* Optimized for future Pro features (Stripe, recurring events, custom forms)

== Shortcodes ==

`[event_register id="123"]`  
Displays the registration form for the event with ID 123.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/arbe-events`
2. Activate the plugin via the Plugins menu in WordPress
3. Create a new Event post
4. Embed the registration form using the shortcode `[event_register id="EVENT_ID"]`
5. Manage registrations via email and/or custom admin views

== Frequently Asked Questions ==

= Can I customize the registration form? =
Not in v1, but Pro version will support custom fields and drag-and-drop form builder.

= Is Stripe integration available? =
Coming in Pro version.

= Can I export registrants? =
CSV export is on the roadmap and will be added soon.

= Is this plugin translation-ready? =
Yes. All strings are translatable and a `.pot` file is included.
